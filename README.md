# NHenCraw-v1
NHenCraw-v1 is a Manga downloader and is NSFW.

## How to use
  1. Download the modules
  ```
      pip3 install -r requirement.txt
  ```
  2. Execute the python script
  ```
      python3 gui.py
  ```
  3. Enter the number from the specific web
